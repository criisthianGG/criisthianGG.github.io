intro CSS by Cristhian García

La hoja de estilo en cascada o CSS es un lenguaje usado para definir la presentación de un documento estructurado que vamos a escribir en html. La idea que se encuentra detrás del desarrollo de CSS es separar la estructura de un documento de su presentación. Las etiquetas aprendidas en khanacademy sobre CSS son las siguientes:

  1. poner color al fondo de una etiqueta o poner color al texto a alguna etiqueta by Cristhian García

    2. lo que tienes que poner dentro de la etiqueta style es la etiqueta que deseas modificar, despues una llave y dentro de ella si quieres modificar el fondo tienes que escribir background-color: (el color que quieras en rgb) y si solo quieres cambiar el color del texto tienes que poner color:(el color que desees en rgb) yo lo he usado en la etiqueta p y h2

  2. poner color al fondo de un ID o poner color al texto a un ID by Cristhian García

    3. en este parrafo como no queremos que sea del mismo color que los demás,usaremos un ID para hacer que esta etiqueta esté con un fondo diferente y un color de letra diferente. para ellos debemos de pòner dentro de la etiqueta style #(el nombre de tu ID)y despues como hicimos en el anterior paso. en este parrafo hemos usado el nombre de ID tic.El problema de esto es que no puedes poner un ID a mas de una etiqueta.

  3. poner color al fondo de una clase o poner color al texto a una clase by Cristhian García

    4. con esto podemos hacer que varios parrafos esten del mismo color y asi solucionar el problema que teniamos con los ID. Para usarlo hay que poner dentro de la etiqueta style .(nombre de la clase) y despues sigue igual que las anteriores. A mi clase la he llamado tic-cgg.
